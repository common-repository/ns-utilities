== Description ==

This plugin extends the category. Allows us to store meta for these. You can associate images with various formats to categories in your admin section.

If you have a new translation for your language, please send us by email and we'll add it to the package.

== Installation ==

1. Upload the contents of the zip file (`ns_utilities folder`) to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Edit the settings under the section 'Settings > NS Utilities' on your Wordpress admin panel.
